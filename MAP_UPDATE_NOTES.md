# 町丁目詳細ページと地図操作

## 変更したファイル

- `app.js`：町丁目詳細ページ、MapLibreのクリック・ダブルクリック、キーボード操作、地図状態の保存と復元、初回操作案内
- `styles.css`：選択中の丁目、詳細ページ、操作案内、スマートフォン表示
- `data.js`：`mapItems` の各デモ情報に `areaId` を設定

`map-geo.js` は町丁目境界、人口、世帯数、面積の元データとして、そのまま利用しています。

## 記事や開発情報を町丁目へ紐づける方法

`data.js` の `mapItems` に `areaId` を入力します。

```js
{
  id: "example-item",
  title: "記事や開発情報のタイトル",
  areaId: "14215001201",
  targetArea: "中央一丁目"
}
```

`areaId` には、`map-geo.js` にある対象町丁目の `Feature.id` を文字列として入力します。画面表示用の町名ではなく、このIDを使って関連付けます。

中央一丁目のIDは `14215001201` です。URLは `#/areas/14215001201` になります。

## 地図から詳細ページを開く方法

- 町丁目を1回クリックまたはタップ：選択した丁目へ拡大し、右側に概要を表示
- 町丁目をダブルクリック：詳細ページを開く
- 右側の「この地域を詳しく見る」：詳細ページを開く
- キーボード：町丁目を選択し、Enterキーまたはボタンで詳細ページを開く

詳細ページへ進む前に、中心座標、ズーム率、選択中の町丁目ID、カテゴリーをブラウザ内へ保存します。「地図に戻る」またはブラウザの戻る操作で、直前の地図状態を復元します。

掲載中の地点、計画、記事はすべて画面確認用のデモデータです。
